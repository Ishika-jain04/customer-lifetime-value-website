# Customer Lifetime Value Website

A modern Customer Lifetime Value (CLV) analytics platform designed to help businesses measure, analyze, and visualize customer value over time. The application provides interactive dashboards, retention insights, cohort analysis, and industry benchmarking for better business decision-making.

## Features

- Customer Lifetime Value (CLV) visualization
- Interactive analytics dashboard
- Cohort analysis
- Industry benchmark comparison
- Customer retention insights
- Responsive and modern user interface
- Smooth animations and interactive charts

## Technologies Used

- TypeScript
- HTML5
- CSS3
- Vite
- Modern JavaScript (ES6+)

## Installation

1. Clone the repository
   ```bash
   git clone https://github.com/your-username/customer-lifetime-value-website.git
   ```

2. Navigate to the project folder
   ```bash
   cd customer-lifetime-value-website
   ```

3. Install dependencies
   ```bash
   npm install
   ```

4. Start the development server
   ```bash
   npm run dev
   ```

## Project Structure

- Interactive dashboard
- CLV growth visualization
- Industry comparison charts
- Customer cohort analysis
- Responsive UI components

## Purpose

This project demonstrates the implementation of a modern web application for customer analytics using reusable components, responsive design, and data visualization techniques.
